# Commands

| Command                 | Description                              |
| ----------------------- | ---------------------------------------- |
| npm docs [package-name] | Open package's documentation page        |
| npm repo [package-name] | Open package's GitHub repo               |
| npm outdated            | Check packages for outdated dependencies |
