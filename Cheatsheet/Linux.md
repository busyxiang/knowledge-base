# Command
### List All Folders
```bash
ls
```
#### Make Directory
```bash
mkrdir <folder_name>
```
#### Delete File
```bash
rm <file_path>
```
#### Delete Folder
```bash
rmdir <folder_path>
```
#### Delete Folder and all its files
```bash
rm -rf <folder_path>
```
# Alias
### List all alias
```bash
alias
```
### Get alias source
```bash
which <command>
```
#### Create Permanent Alias
Edit  `~/.bash_aliases` or `~/.bashrc`  file
```bash
alias <name>='<command>'
```
