# Command
## Create new branch and switch to it
```bash
git checkout -b <branch>
```
## Discard existing files uncommitted changes
```bash
git reset --hard
```
## Remove new and uncommitted files
```bash
git clean -fxd
```
## Switch local branch
```bash
git switch <branch>
```
## Push to another branch
```bash
git push origin <source>:<target>
```
