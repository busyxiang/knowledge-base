# Modes
| Command | Description         |
| ------- | ------------------- |
| i       | insert              |
| :       | command             |
| v       | visual              |
| esc     | exit to normal mode |

# Command
### Quit
```bash
:q
```
### Quit and Discard Changes

```bash
:q!
```
### Save
```bash
:w
```
### Save and Quit
```bash
:wq
```
