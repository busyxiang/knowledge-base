# Modes
| Key | Description         |
| --- | ------------------- |
| i   | insert              |
| :   | command             |
| v   | visual              |
| esc | exit to normal mode |

# Command
| Command | Description              |
| ------- | ------------------------ |
| :q      | Quit                     |
| :q!     | Quit and discard changes |
| :w      | Save                     |
| :wq     | Save and Quit                         |
